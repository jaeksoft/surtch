### Segment files:

 - {field}.fst : Map of Term/Idx
 - {field} dox : Array of DocOffset (Fixed size)
 - {field}.doc: : Array of DocIds (dynamic bitset)
 - {field}.pox: : Array of PositionsOffsets (Fixed size)
 - {field}.pos: : Array of Positions (dynamic arrays)
